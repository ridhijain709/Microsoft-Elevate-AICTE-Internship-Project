# 🚀 India Vocational Training Intelligence Hub 
### Microsoft Elevate AICTE Internship Capstone Project

## 📊 Project Overview
This project transforms fragmented vocational training data (2015-2021) into a **High-Tech Strategic Dashboard**. The goal is to ensure that every student in India has access to a training seat by identifying where demand exceeds infrastructure.

## 🔴 The Problem (The Strategic Challenge)
Government and educational leaders faced a **"Resource Blind Spot."** With thousands of training centers across India, there was no way to visualize the relationship between:
1. **Enrollment Velocity:** How fast are students signing up?
2. **Infrastructure Lag:** Is the seating capacity growing fast enough?
3. **Regional Disparity:** Which districts are underserved?

Without this dashboard, resources were being allocated based on "guesses" rather than "data," leading to overcrowded centers in some districts and empty seats in others.

## 🛠️ The "Hitech" Solution
I engineered an interactive **Business Intelligence Suite** that features:
- **Executive KPI Bar:** Instant visibility into Total Trainees, Capacity, and Utilization rates.
- **Geospatial Intelligence:** A dynamic map allowing users to hover over any state to see a 5-year trend pop-up (**Report Page Tooltip**).
- **Hierarchical Treemaps:** Visualizing district-level density to find "Success Hotspots."
- **Temporal Slicers:** A professional "Tile-style" year selector to watch the Skill India mission evolve.

## ⚙️ Technology & Logic
- **Tool:** Microsoft Power BI
- **Data Cleaning:** Power Query (Standardized district names and handled null values).
- **Advanced DAX:** Created measures for `Utilization %` and `Trainee-to-Seat Ratio`.
- **Navigation:** Used **Bookmarks** to create a seamless, app-like user experience.

## 📈 Key Insights
- **The 80/20 Rule:** Identified that 20% of districts drive 80% of total enrollment.
- **Utilization Gap:** Found that while national capacity is high, specific states are operating at over 95% capacity, requiring immediate infrastructure expansion.

## 📁 How to Access
1. Download the `Project_Dashboard.pbix` file.
2. Open in **Power BI Desktop**.
3. Use **Ctrl + Click** on the top navigation buttons to switch views.
